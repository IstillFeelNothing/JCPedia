About JCPedia

Цей проект створений для демонстрації навичок у розробці веб-сайтів, зокрема в HTML та CSS. Він присвячений японським автомобілям і містить інформацію про різні бренди та моделі. Мета проекту - навчитися створювати веб-сайти за допомогою HTML та CSS.

Mission

  • Використати як можна більше різних HTML та CSS елементів
  • Попрактикуватися в створенні адаптивних веб-сайтів
  • Попрактикуватися в створенні dropdown меню
  • зробити "нормальний" дизайн
